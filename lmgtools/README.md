# LMG95 Power Logger

Log measurements from a ZES Zimmer LMG95 precision power analyzer into Home
Assistant via MQTT (with auto-discovery) and/or InfluxDB.

Connect the LMG95 through an RS232-Ethernet converter, point the add-on at it,
and the readings — power, voltage, current, frequency and more — show up as
Home Assistant sensors.

See the **Documentation** tab for setup and configuration details.
